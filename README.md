## Design Style

The UI follows a **Neo-Brutalism** design style — strong color contrast, bold borders, hard drop shadows, and clear typography.

---

## Notes

- All screens are static mockups, no API calls or real data.
- Screenshots are located in the `screenshot/` folder at the root of this branch.
- This branch will be used as a visual reference for the main development branch.
